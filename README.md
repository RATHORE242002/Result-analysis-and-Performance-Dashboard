# Result-analysis-and-Performance-Dashboard

A web-based application built using Django, Pandas, Bootstrap, and Chart.js to manage and analyze student results from CSV/Excel files.

🚀 Features

📂 File Management

Upload CSV and Excel (.xlsx) files
View uploaded file data in table format
Search and filter data

🎓 Student Result System

Search student by Roll Number
Display subject-wise marks
Show Pass/Fail status automatically

📊 Data Visualization

Bar Chart (Marks per subject)
Line Chart (Performance trend)
Pie Chart (Pass vs Fail)
Doughnut Chart (Marks distribution)
Average Marks Chart
Performance Analysis (Excellent / Good / Poor)

🔐 Authentication

Login required for secure access

🛠️ Technologies Used

Backend: Django (Python)
Frontend: HTML, CSS, Bootstrap
Data Processing: Pandas
Charts: Chart.js
Database: SQLite (default)





Role-based restriction (Admin / Faculty)
